# Nekdrop
Nekdrop is a Zero-Trust file relay system designed for a high-security environments, allowing users to bridge the gap between untrusted public terminals and private mobile nodes. 
# Why am i doing this? 
Basically my motivation behind this is to share file from college's computer to my mobile without opening any third party software or webapp. 
In my college you have to make files and to make those files you need screenshots of the program that are done in the college's system. 
The shareing the images is usually done through signing in you gmail, or whatsapp, which for me is a big security risk. The Nekdrop is aiming to solve this problem, making such file sharing easier.
